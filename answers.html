Part C: Short Answer

  1. Explain the difference between a semantic element (e.g., nav) and a generic element (e.g., div).

    A semantic element (such as <nav>) conveys the meaning and purpose of its content to browsers, screen readers, and search engines, improving accessibility and structure. Whereas a generic element (such as <div>) has no inherent meaning and is used only for grouping or styling content, providing no information about the role of that content.

  2. Why does KLM not model high-level planning and interpretation?

    KLM does not include high-level planning and interpretation because it assumes the user already knows what to do. Since the task is routine and error-free, KLM only measures the time for simple actions, not thinking or decision-making.
